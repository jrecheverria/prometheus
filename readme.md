simple developer environment initialization script.

## Brings in

- **homebrew package manager**
- **ghostty terminal emulator + config**
- **github cli**
- **neovim**
- **ripgrep + fd (for telescope)** - ultra fast search
- **kickstart.nvim** - minimal neovim starter with sane defaults
- **oil.nvim** - minimal file explorer
- **toggleterm.nvim** - persistent terminal that works like vscode's integrated terminal

## Run

```sh
./init_fire.sh
```
